Proyecto: Agencia de viajes Travel World, desarrollado grupalmente en Java con componentes web de html y jsp.

En el comprimido habran dos carpetas sumados a este achivo readme.txt
Una de las carpetas se llamara librerias, donde estaran incluidas las librerias externas que requirio el proyecto.
Cada que se habra el programa en un directorio nuevo, se desconfiguraran las librerias por lo cual:
	- Se debe ir a netbeans con el proyecto abierto, click derecho sobre el proyecto y seleccionar en la
	parte inferior en la opcion de reparar enlaces perdidos
	-Apareceran todos los archivos mal direccionados, que son librerias del proyecto, dar clic sobre cada una
	y seleccionar las que tienen el mismo nombre en la carpeta librerias que se menciono anteriormente.
	-Repetir hasta que ya no hayan mas problemas de directorio.
De ser necesario, se sugiere el server apache tomcat 8.35, que fue sobre el cual se desarrollo el programa
El archivo principal del programa o por dodne empieza es el index.java, para ejecutar el proyecto solo dar clic derecho sobre
el proyecto y Run.

La herencia es muy basica y se presenta entre las clases persona y cliente.
Muchas funcionalidades quicimos implementar pero no fue posible por el tiempo. Se implemento el aviso al correo cuando se registrara
pero o se bloqueo el correo o demora mucho, pues despues de 40 minutos o mas llegan normalmente.
